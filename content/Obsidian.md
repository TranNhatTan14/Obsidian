### Sample Python Code

Here is an example of a Python function that calculates the factorial of a number:

```python
def factorial(n):
    """Return the factorial of a number."""
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

# Example usage
print(factorial(5))  # Output: 120
```
